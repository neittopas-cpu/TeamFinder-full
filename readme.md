# TeamFinder — платформа для поиска команды проектов

## Описание проекта

TeamFinder – это веб-приложение для поиска единомышленников и совместной работы над Open Source проектами.  
Пользователи могут:

- создавать проекты с указанием GitHub-репозитория;
- присоединяться к чужим проектам в качестве участников;
- добавлять проекты в избранное;
- просматривать профили других участников;
- фильтровать участников по различным критериям.

Проект реализован на Django с использованием кастомной модели пользователя, генерации аватаров, пагинации и **SQLite** (для простоты локального запуска).

## Используемые технологии

- Python 3.10+
- Django 5.2
- SQLite (по умолчанию, не требует отдельного сервера)
- Pillow (генерация аватаров)
- python-decouple (опционально, можно без него)
